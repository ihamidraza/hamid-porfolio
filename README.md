## üì¶ Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ihamidraza/hamid-porfolio.git
cd hamid-porfolio
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üé® Customization

### Personal Information

Update your personal information in the following files:

- `app/layout.tsx` - Update metadata (title, description, etc.)
- `components/Hero.tsx` - Update name, title, and social links
- `components/About.tsx` - Update about section content
- `components/Experience.tsx` - Add your work experience
- `components/Skills.tsx` - Update skills and proficiency levels
- `components/Projects.tsx` - Add your projects
- `components/Contact.tsx` - Update contact information

### Colors and Styling

The design uses a gradient color scheme. You can customize colors in:

- `app/globals.css` - Update CSS variables and utility classes
- Tailwind classes throughout components

## üöÄ Deployment

### Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Vercel will automatically detect Next.js and deploy

### Build for Production

```bash
npm run build
npm run start
```
